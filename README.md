# Блок-схема программы мониторинга пиковой нагрузки портала

## Описание программы
Программа суммирует **N интервалов времени** (где 10 ≤ N ≤ 30) и вычисляет **общее время работы портала**.  
Пользователь вводит количество интервалов, затем значения каждого интервала, после чего программа выводит итоговую сумму.

---

## Алгоритм работы программы
1. Ввести число **N** — количество интервалов (10 ≤ N ≤ 30).
2. Проверить, что N находится в допустимом диапазоне.
3. Инициализировать переменную `total = 0`.
4. Для каждого из N интервалов:
   - считать значение интервала,
   - добавить его к `total`.
5. Вывести итоговое значение `total` — общее время работы портала.
6. Завершить работу программы.

---

```mermaid
